# tools.fileformat.info

These are some useful tools for viewing, analyzing and fixing file formats.  They are web-based, but do all processing locally in your browser.

[![deploy](https://github.com/FileFormatInfo/tools.fileformat.info/actions/workflows/ghpages-deploy.yaml/badge.svg)](https://github.com/FileFormatInfo/tools.fileformat.info/actions/workflows/ghpages-deploy.yaml)

## Tools

* [Byte count](https://tools.fileformat.info/bytecount.html) - count the byte values in a file
* [Rune count](https://tools.fileformat.info/runecount.html) - count the characters (UTF-8, UTF-16 or Latin-1) in a file
